# Contributing

Obrigado por contribuir! Siga estes passos:

1. Fork e clone o repositório
2. Crie uma branch a partir de `dev` com o formato `feat/<issue-number>` ou `fix/<issue-number>`
3. Abra um PR direcionado para `dev` com descrição clara e checklist
4. Execute os testes locais antes de submeter

Formato de commits: `type(scope): descrição` (Ex.: `feat(readme): adicionar seção de instalação`)
