# GitHub Project da Sprint 1

Este projeto organiza a Sprint 1 do SimplesEditor em formato Kanban.

## Projeto

- **Nome:** SimplesEditor Sprint 1
- **URL:** https://github.com/users/GrouwBer/projects/1
- **Escopo:** acompanhar as issues e PRs da base do repositório sem bloquear o restante do time.

## Colunas

O board usa o campo **Status** como coluna:

- **Todo**: item ainda não iniciou ou aguarda triagem.
- **In Progress**: item em execução.
- **Done**: item concluído e revisado.

## Automações

- Itens novos entram em **Todo**.
- Itens fechados ou PRs mesclados vão para **Done**.
- Itens reabertos retornam para **Todo**.

## Uso

1. Adicione a issue ou PR ao projeto.
2. Mantenha o status atualizado durante o trabalho.
3. Mova o item para **Done** quando a entrega estiver publicada.
